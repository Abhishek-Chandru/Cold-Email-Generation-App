# Cold Email Generator (AI-Powered)

An AI-powered web app that generates personalized cold emails for job applications.  
Upload your resume + paste/upload a Job Description, and the app automatically:

- Extracts job details  
- Reads and understands your resume  
- Detects the most relevant resume points  
- Writes a tailored, professional cold email  

This app is built using **Streamlit**, **LangChain**, and **Groq LLaMA 3.1**.
